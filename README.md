Proyectos Destacados
Estos son los proyectos más representativos en los que he trabajado.

1. GestiFleet: Plataforma de Gestión de Flotas (Full-Stack)
Descripción: Aplicación web full-stack (Práctica UCM) para la gestión de flotas de vehículos eléctricos corporativos.

- Stack: Node.js, Express.js, MySQL, JavaScript, Bootstrap.

- Características: API RESTful con RBAC , carga de datos desde JSON  y un fuerte enfoque en accesibilidad (a11y).   

- Estado: En desarrollo. El código se mantiene privado por política académica hasta su defensa oral.

2. Ecosistema de Smart Contracts (Web3)
Descripción: Proyecto de autoaprendizaje para la creación y gestión de un token fungible (ERC-20) y un NFT (ERC-721).

- Stack: Solidity, Python, Flask, Web3.py, Ganache.

- Características: Incluye una dApp básica (Flask) para interactuar con los contratos desplegados en un entorno local.

- Repositorio: Smart-Contracts

3. LaTina: Gestor de Turnos y Nóminas (JavaFX)
Descripción: Aplicación de escritorio avanzada (JavaFX) que utiliza vistas web (HTML/CSS/JS) para la gestión de turnos y personal.

- Stack: Java, JavaFX, HTML5, CSS3, JavaScript, Jira, Scrum.

- Características: Frontend en WebView con validación (JS), lógica de negocio en Java, y gestión ágil del proyecto con Scrum y Jira.

- Repositorio: LaTina

4. AirGestRSP: Arquitectura de Software
Descripción: Aplicación de gestión aeroportuaria desarrollada en un equipo de 12 personas, cubriendo el ciclo de vida completo del software.

- Stack: Java, MySQL, JPA, JUnit, UML.

- Características: Diseño de arquitectura, modelado UML (IBM) y desarrollo de persistencia (JPA) y pruebas unitarias (JUnit).

- Repositorio: AirGestRSP

5. Optimización (Programación con Restricciones)
Descripción: Proyectos académicos (UCM) para resolver problemas de optimización complejos usando modelado declarativo y solvers SMT.

- Stack: MiniZinc, Python, Z3 (SMT Solver).

- Características: Incluye un planificador de turnos (MiniZinc) y un optimizador de producción de aceite (Python/Z3).

- Repositorio: PR
